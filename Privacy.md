# Privacy Policy for MustLearnKQL

Last updated: January 30, 2026

## Introduction

This Privacy Policy applies to the MustLearnKQL GitHub repository (https://github.com/rod-trent/MustLearnKQL), which provides educational resources, code examples, queries, datasets, and related materials for learning the Kusto Query Language (KQL). The repository is intended for educational purposes and does not involve any direct collection, processing, or storage of personal data from users.

If you interact with the repository (e.g., by cloning, forking, or contributing), your activities are governed by GitHub's Privacy Statement (available at https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement). This policy focuses specifically on the MustLearnKQL project.

## Data Collection

- **No Personal Data Collected**: This repository does not collect, track, or store any personal information from users. It consists of static files, sample datasets, documentation, and educational content. There are no forms, analytics tools, or scripts embedded in the repository that gather user data.
  
- **Sample Datasets**: Any datasets provided in the repository (e.g., in the `Datasets/` folder) are synthetic or anonymized sample data for learning purposes only. They do not contain real personal information.

## Data Usage

Since no personal data is collected, there is no usage, sharing, or processing of such data by this project.
